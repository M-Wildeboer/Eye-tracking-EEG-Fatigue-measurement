# Eye-tracking-EEG-Fatigue-measurement

A repository for usage of the YETI eye tracker combined with EEG data.

Based on [https://github.com/amirashv/YET/tree/main/yeti24](https://github.com/amirashv/YET/tree/main/yeti24) for the YETI eye tracker code and 3d-models of the clips for the headframe.
The extra used 3d-models (the dopper waterbottle base and adapted barebone clip) are added in this repository, the other models can be found in the reference git-reporitory.
